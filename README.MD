本仓库是图书 《OpenClaw实战：智能体系统工程》 的配套资源仓库，用于提供与本书相关的学习资料、示例资源以及读者交流入口。

本仓库的目标是帮助读者更好地理解 AI 智能体系统（AI Agent Systems） 的设计方法，并探索如何基于 OpenClaw 构建真实可运行的智能体系统。

关于本书

随着大语言模型的发展，人工智能正从简单的对话系统逐渐走向能够 规划任务、调用工具、持续执行并与数字环境交互的智能体系统。

本书以 OpenClaw 为实践案例，系统介绍如何构建可靠的智能体系统工程，包括：

Agent 系统架构设计

Agent 执行循环（Agent Loop）

上下文与记忆管理

工具调用与执行环境

多智能体系统协作

Agent 系统的安全与边界

本书重点关注 如何将大模型能力真正落地为可运行的智能体系统。
